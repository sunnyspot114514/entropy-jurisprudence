# 判例演化实验

[🇺🇸 English](README.md) | [🇨🇳 中文说明](README.zh-CN.md)

## 概述

本实验模拟 AI "文明"如何随时间发展法律判例。AI 扮演最高法院角色，在尊重历史判决的前提下做出新裁决。

## 核心概念

与主实验（单案例评估）不同，本实验探索：
- **判例记忆**：AI 记住并引用过去的判决
- **一致性压力**：必须解释与历史的偏离
- **法律演化**：道德标准如何随迭代漂移

## 使用方法

```bash
python precedent_evolution.py
```

## 输出文件

| 文件 | 描述 |
|------|------|
| `common_law_db.txt` | 累积判例 |
| `precedent_*.json` | 实验快照 |

## 测试案例

1. **举报人** - 揭发犯罪但违反保密协议
2. **偷药救人** - 医生偷药救孩子
3. **AI 附带损害** - AI 删病毒时误删用户数据

## 与主实验的关系

| 方面 | 主实验 | 本实验 |
|------|--------|--------|
| 关注点 | 公式一致性 | 历史一致性 |
| 记忆 | 无状态 | 有状态 |
| 目标 | 测试 E=H×R 逻辑 | 测试判例遵循 |
